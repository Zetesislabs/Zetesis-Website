# Zetesis Labs Website

Minimal, academic-styled website for Zetesis Labs.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Vanilla CSS + CSS Variables (No Tailwind)
- MDX for content

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure
- `app/`: App router pages (`main`, `lab`, `dhruv`).
- `components/`: Shared React components (NavBar, TeamTable, Gallery, etc.).
- `content/`: MDX files and YAML data.
- `public/`: Static assets (images, logos).
